# 🛡️ Cybersecurity Automation

A collection of cybersecurity automation workflows built with **n8n** to automate Security Operations Center (SOC) tasks, incident response, and security reporting.

---

## 🚀 Current Project

### 🔐 Security Incident Triage & Response Playbook

An end-to-end security incident response workflow that automates the incident handling process from intake to reporting.

### Features

- ✅ Security Incident Intake Form
- ✅ Data Validation & Enrichment
- ✅ Automated Risk Scoring
- ✅ MITRE ATT&CK Technique Mapping
- ✅ SOC Knowledge Base
- ✅ Incident Prioritization
- ✅ Google Sheets Logging
- ✅ HTML Report Generation
- ✅ Automated Email Notifications

---

## 🔄 Workflow

```text
Security Incident Form
        │
        ▼
Validate & Enrich
        │
        ▼
SOC Knowledge Base
        │
        ▼
Google Sheets
        │
        ▼
HTML Report Generator
        │
        ▼
Email Report
```
